## <span class="privacy-header-firefox"></span> <span class="privacy-header-policy">Уведомление о конфиденциальности для веб-браузера Firefox</span>

*Дата вступления в силу: 28 сентября 2017 г.*
{: datetime="2017-09-28" }

## Компания Mozilla считает, что обеспечение конфиденциальности является фундаментальным требованием для полноценной работы в Интернете.

Для выполнения этого требования мы встроили в браузер Firefox, как и во все другие наши продукты, средства, с помощью которых пользователи могут в широких пределах контролировать информацию, которую они распространяют в Интернете и которой делятся с нами. Мы стремимся собирать только те сведения, которые необходимы нам, чтобы улучшить работу браузера Firefox для всех пользователей.

В настоящем Уведомлении о конфиденциальности мы объясняем, какие данные собирает приложение Firefox и какие настройки вы можете использовать, чтобы ограничить объем этих данных. Кроме того, все процедуры получения, обработки и передачи информации в браузере Firefox соответствуют [Политике конфиденциальности](https://www.mozilla.org/privacy/) компании Mozilla.

## По умолчанию Firefox передает данные для достижения следующих целей:

### Повышение быстродействия и стабильности браузера для всех пользователей {: #health-report }

* __Данные о взаимодействии__: Firefox отправляет нам данные о вашем взаимодействии с этим браузером (количество открытых вкладок и окон, количество посещенных страниц, число и тип установленных дополнений, продолжительность сеансов и т. д.) и об использовании функций Firefox, предлагаемых компанией Mozilla или нашими партнерами (таких, как поиск Firefox и поиск по партнерским ссылкам).  

* __Технические данные__: Firefox отправляет нам данные о версии и языке браузера, операционной системе устройства и конфигурации оборудования, объеме памяти, сбоях и ошибках, результатах автоматизированных процессов, таких как обновление, безопасный браузинг или активация.  Когда Firefox отправляет нам данные, временно передается и ваш IP-адрес (как элемент журналов нашего сервера).  

Ознакомьтесь с документацией о телеметрии для [ПК](http://firefox-source-docs.mozilla.org/toolkit/components/telemetry/telemetry/index.html)/[Android](http://firefox-source-docs.mozilla.org/mobile/android/fennec/index.html)/[iOS](https://github.com/mozilla-mobile/firefox-ios/wiki/Telemetry) или узнайте, как [отказаться](https://support.mozilla.org/kb/send-performance-data-improve-firefox) от сбора этих данных.
{: #telemetry }

### Установка поисковой системы по умолчанию {: #defaultsearch }

* __Данные о местоположении__:  Когда вы впервые используете Firefox, ваш IP-адрес используется для установки поисковой системы по умолчанию в зависимости от вашей страны.  [Подробнее](https://support.mozilla.org/kb/change-your-default-search-settings-firefox).

### Предложение соответствующего контента 

Firefox отображает определенный контент, например, «фрагменты» (сообщения от Mozilla) и основные сайты (сайты, предлагаемые компанией Mozilla для тех, кто использует Firefox впервые).

* __Данные о местоположении__: Firefox использует ваш IP-адрес, чтобы предложить контент, соответствующий вашей стране.

* __Технические данные и сведения о взаимодействии__: Firefox отправляет нам данные о положении, объеме и размещении предлагаемого нами контента, а также основные сведения о вашем взаимодействии с этим контентом. Сюда входит информация о том, сколько раз предложенный контент отображался и сколько ссылок в нем было открыто.

* __Данные веб-страницы для фрагментов__: Когда вы нажимаете ссылку фрагмента, мы можем получать данные о ссылке, на которую вы перешли. Эти данные не связаны с какой-либо другой информацией о вас. [Подробнее](https://abouthome-snippets-service.readthedocs.io/en/latest/data_collection.html).
{: #snippets }

### Повышение уровня безопасности для всех пользователей {: #security }

* __Технические данные для обновлений__: Версия Firefox для ПК периодически проверяет наличие обновлений путем подключения к серверам Mozilla. Для корректного обновления требуются сведения о версии Firefox, языке и операционной системе устройства. Мобильные версии Firefox могут подключаться к другой службе, если вы использовали ее для загрузки и установки этого браузера. [Подробнее](https://support.mozilla.org/kb/how-stop-firefox-automatically-making-connections#w_auto-update-checking).
{: #auto-updates }

* __Технические данные для занесения нежелательных дополнений в черный список__: Браузеры Firefox для ПК и Android периодически подключаются к службам Mozilla для защиты пользователей от вредоносных дополнений.  Для обновления черного списка ваших дополнений требуются сведения о версии Firefox, языке, операционной системе устройства и установленных дополнениях. [Подробнее](https://blog.mozilla.org/addons/how-to-opt-out-of-add-on-metadata-updates/). 

* __Получение сведений о веб-странице и технических данных для службы SafeBrowsing Google__: Чтобы защитить вас от вредоносного контента, браузер Firefox отправляет основную информацию о загрузках нераспознанных файлов (включая имя файла и URL-адрес, с которого он был загружен) в службу SafeBrowsing Google.  

    [Узнайте подробности](https://support.mozilla.org/kb/how-does-phishing-and-malware-protection-work) или ознакомьтесь с [Политикой конфиденциальности Google](https://www.google.com/policies/privacy/). Отказ от этой опции приводит к отключению предупреждений Firefox о потенциально опасных веб-сайтах и файлах.

* __Получение сведений о веб-странице и технических данных для служб сертификации__: Когда вы посещаете безопасный веб-сайт (обычно URL-адрес такого сайта начинается с HTTPS), Firefox подтверждает его [сертификат](https://support.mozilla.org/kb/secure-website-certificate). Для этого браузер Firefox может отправлять определенную информацию о веб-сайте в центр сертификации, указанный этим веб-сайтом.  

    Отказ от этой опции увеличивает риск перехвата вашей персональной информации. [Подробнее](https://support.mozilla.org/kb/advanced-settings-browsing-network-updates-encryption#w_certificates-tab).

### Отчеты о сбоях {: #crash-reporter }
В версиях Firefox для ПК по умолчанию мы попросим вас предоставить специалистам компании Mozilla отчет с подробной информацией о сбоях, но вы всегда можете отказаться от отправки отчета.

* __Конфиденциальные данные__:  Отчет о сбое включает файл дампа памяти Firefox на момент отказа. Этот файл может содержать ваши личные данные или другую важную для вас информацию.

* __Данные веб-страницы__:  Отчет о сбое включает активный URL-адрес на момент отказа.

* __Технические данные__:  Отчет о сбое включает в себя сведения о причине отказа Firefox, состоянии памяти устройства и выполняемых операциях во время отказа.

Ознакомиться с полной документацией можно [здесь](https://firefox-source-docs.mozilla.org/toolkit/crashreporter/crashreporter/index.html).

### Поддержка наших маркетинговых кампаний и оценка их эффективности

* __Данные о кампании и направленных клиентах__: Эти данные помогают сотрудникам Mozilla оценить эффективность наших маркетинговых кампаний.
{: #referraltracking }

    _Для ПК_: По умолчанию Firefox отправляет компании Mozilla HTTP-данные, которые могут быть включены в программу установки Firefox.  Это позволяет идентифицировать домен сайта или рекламную кампанию (если таковая имеется), которые направили вас на нашу страницу загрузки. Ознакомьтесь с [полной документацией](https://firefox-source-docs.mozilla.org/toolkit/components/telemetry/telemetry/data/environment.html#attribution) или [откажитесь от этой опции](https://support.mozilla.org/kb/desktop-privacy) before installation.

    _Для устройств на базе iOS и Android_: Firefox по умолчанию отправляет данные мобильной кампании нашему поставщику аналитических решений Adjust, который руководствуется собственной [политикой конфиденциальности](https://www.adjust.com/privacy_policy/).  Данные мобильной кампании включают рекламный идентификатор Google, IP-адрес, временную метку, а также сведения о стране, языке, региональных настройках, операционной системе и версии приложения. Ознакомьтесь с [полной документацией](https://firefox-source-docs.mozilla.org/mobile/android/fennec/adjust.html).
{: #thirdparty }

* __Технические данные и сведения о взаимодействии__: 

    _Для устройств на базе iOS и Android_: По умолчанию Firefox отправляет сведения о том, какие функции вы используете в браузере, нашему поставщику решений для мобильного маркетинга Leanplum, который руководствуется собственной [политикой конфиденциальности](https://www.leanplum.com/privacy/).  Эти данные позволяют нам тестировать различные функции и интерфейсы, а также предоставлять индивидуальные сообщения и рекомендации пользователям для повышения удобства работы с Firefox.

    Ознакомьтесь с документацией для [iOS](https://github.com/mozilla-mobile/firefox-ios/blob/master/MMA.md)/[Android](https://firefox-source-docs.mozilla.org/mobile/android/fennec/mma.html) или узнайте, как [отключить эту функцию](https://support.mozilla.org/kb/send-anonymous-usage-data-firefox-mobile-devices).

## Если вы используете эти функции, Firefox будет передавать данные для реализации следующих возможностей: {: #optional-features }

### Поиск

Вы можете выполнять поиск непосредственно из нескольких мест в браузере Firefox, в том числе в адресной строке, в строке поиска и на новой вкладке.  _Mozilla не получает сведения о ваших поисковых запросах._Данные запроса отправляются вашему поставщику поисковой системы, который имеет собственную политику конфиденциальности.

* __Поисковые предложения__: Firefox по умолчанию посылает поисковые запросы поставщику поисковой системы, чтобы помочь вам найти общие фразы, которые использовали для поиска другие люди, и улучшить таким образом результаты поиска. Если вы отключите функцию поисковых предложений в своей поисковой системе, эти данные не будут отправляться.
{: #searchsuggestions } 

    [Ознакомьтесь с дополнительной информацией](https://support.mozilla.org/kb/use-popular-search-suggestions-firefox-search-bar), включая сведения о том, как отключить эту функцию.

### Аккаунты Firefox

* __Данные аккаунтов Firefox__: При создании аккаунта Firefox компания Mozilla получает сведения о вашем адресе электронной почты и хэше пароля.  По своему выбору вы можете включить отображаемое имя и аватар.  Информация о вашем адресе электронной почты отправляется нашему поставщику почтовых услуг SalesForce Marketing Cloud, который руководствуется собственной [политикой конфиденциальности](https://www.marketingcloud.com/privacy-policy/website-privacy-statement/). Если вы используете свой аккаунт Firefox для доступа к другим веб-сайтам или службам (таким как AMO или Pocket), мы получаем временную метку вашего входа в интерфейс этих служб. 

* __Данные о местоположении__: В целях безопасности мы сохраняем IP-адрес, который вы используете для доступа к аккаунту Firefox, чтобы приблизительно определить ваш город и страну.  Эти данные используются для отправки вам оповещений по электронной почте в случае обнаружения подозрительных действий, таких как вход в аккаунт из других мест.

* __Данные о взаимодействии__: Мы получаем сведения о посещениях веб-сайта для регистрации аккаунтов Firefox и взаимодействии с ним, а также о настройках меню и обмене начальными сообщениями, электронными письмами и SMS. [Ознакомьтесь с дополнительной информацией](https://www.mozilla.org/privacy/websites/) о методах, используемых компанией Mozilla для обработки данных веб-сайтов и электронной почты.  

* __Технические данные__: Для отображения устройств, которые синхронизируются с вашим аккаунтом Firefox, и поддержки дополнительных функций, мы сохраняем сведения об операционной системе, браузере и его версии, временной метке, языке и региональных параметрах вашего основного устройства и всех других устройств, подключенных к вашему аккаунту.  

Ознакомьтесь с [полной документацией](https://github.com/mozilla/fxa-auth-server/blob/master/docs/metrics-events.md) или [получите дополнительную информацию](https://support.mozilla.org/kb/access-mozilla-services-firefox-accounts), включая сведения о том, как [удалить ваш аккаунт](https://support.mozilla.org/kb/how-do-i-delete-my-firefox-account).

### Синхронизация {: #sync }

* __Синхронизация данных__: Если вы включите синхронизацию, компания Mozilla будет получать в зашифрованном виде всю информацию, которую вы хотите синхронизировать на всех устройствах. Сюда могут входить сведения о вкладках, дополнениях, паролях, закладках, истории и настройках Firefox, а также данные для автозаполнения платежных реквизитов.  Удаление аккаунта Firefox приведет к удалению всего контента, предназначенного для синхронизации. Вы также можете ознакомиться с полной [документацией](http://moz-services-docs.readthedocs.io/en/latest/sync/).
  
* __Технические данные и сведения о взаимодействии__: Если вы включите синхронизацию, Firefox будет периодически отправлять с помощью телеметрии основную информацию о последней попытке синхронизации. Эта информация может включать сведения о том, когда выполнялась синхронизация, была ли она успешной или неудачной, и какие типы устройств синхронизировались. Вы также можете ознакомиться с полной [документацией](https://firefox-source-docs.mozilla.org/toolkit/components/telemetry/telemetry/data/sync-ping.html).

[Получите более подробную информацию](https://support.mozilla.org/kb/how-do-i-set-sync-my-computer), включая сведения о том, как включить или отключить синхронизацию.

### Местоположение {: #location-services }

* __Данные о местоположении для службы геолокации Google__: Firefox всегда запрашивает ваше разрешение перед определением вашего местоположения и передачей этих данных запрашивающему веб-сайту (например, когда веб-сайт, отображающий карту, запрашивает ваше местоположение, чтобы показать направление движения).  Для определения местоположения Firefox может использовать функцию геолокации вашей операционной системы, сети Wi-Fi, передатчики сотовых сетей или IP-адрес. Эти данные могут отправляться в службу геолокации Google, который имеет свою собственную [политику конфиденциальности](https://www.google.com/privacy/lsf.html).

[Подробнее](https://www.mozilla.org/firefox/geolocation/).

### Скриншоты Firefox {: #screenshots }

* __Загрузка скриншотов__: Скриншоты, которые вы решили загрузить, отправляются в компанию Mozilla и хранятся там ограниченное время, которое вы можете изменить.  Мы можем получать доступ к загруженным вами скриншотам при возникновении служебной необходимости или для оказания услуг.  Вы можете в любое время удалить загруженные вами скриншоты.  

* __Данные о взаимодействии__: Мы получаем данные о посещениях веб-сайта скриншотов Firefox, периодичности доступа к загруженным скриншотам и периодичности их загрузки вами или другими пользователями, а также об использовании вами кнопок, элементов мозаичного изображения и мыши для создания скриншотов.

    Типы данных, которые мы можем дополнительно собирать при посещениях веб-сайта скриншотов Firefox, описаны в нашем [уведомлении о конфиденциальности для веб-сайтов.](https://www.mozilla.org/privacy/websites/)

* __Технические данные__: Мы получаем данные о среднем объеме и количестве загруженных скриншотов, версии вашего браузера Firefox, операционной системе устройства и ошибках.  IP-адрес, с которого осуществляется доступ к веб-сайту скриншотов Firefox, временно сохраняется (как элемент стандартного журнала сервера). 

Ознакомьтесь с [полной документацией](https://github.com/mozilla-services/screenshots/blob/master/docs/METRICS.md) или [получите дополнительную информацию](https://wiki.mozilla.org/Firefox/Screenshots/FAQs).

### Уведомления веб-сайта {: #push-notifications }

* __Данные подключения__: Если вы разрешите веб-сайту отправлять оповещения, Firefox будет подключаться к службам Mozilla и использовать ваш IP-адрес для передачи сообщений.  Компания Mozilla не может получать доступ к содержимому сообщений.    

* __Данные о взаимодействии__: Мы получаем совокупные данные о количестве оформленных/отмененных подписок Firefox на уведомления веб-сайтов, количестве отправленных сообщений, временных метках и отправителях (эти данные могут включать сведения о конкретных провайдерах веб-сайтов).  

Ознакомьтесь с[полной документацией](https://mozilla-push-service.readthedocs.io/en/latest/) или [получите более подробную информацию](https://support.mozilla.org/kb/push-notifications-firefox), включая сведения о том, как отменить уведомления веб-сайта.

### Дополнения {: #addons }

Вы можете установить дополнения на сайте addons.mozilla.org (АМО) или с помощью диспетчера дополнений Firefox, который доступен в меню на панели инструментов Firefox.
 
* __Поисковые запросы__: Поисковые запросы в диспетчере дополнений отправляются в компанию Mozilla, чтобы предоставить вам с рекомендуемые дополнительные компоненты.  

* __Данные о взаимодействии__:  Мы получаем сводные данные о посещениях сайта АМО и доступе к диспетчеру дополнений в Firefox, а также об использовании контента на этих страницах. Вы можете получить дополнительную информацию о методах обработки данных на [веб-сайтах Mozilla](https://www.mozilla.org/privacy/websites/).

* __Технические данные для обновлений__: Firefox периодически подключается к службам Mozilla, чтобы установить обновления дополнительных компонентов.  Для корректного обновления установленных дополнительных компонентов требуются сведения о версии Firefox, языке и операционной системе устройства.

### Сноска

Настоящее уведомление о конфиденциальности применимо для последней общей версии браузера Firefox, предоставляемой компанией Mozilla.  Если вы получили приложение Firefox из другого источника или используете устаревшую версию, ваша копия Firefox может иметь другие параметры конфиденциальности.  
{: #pre-release }

Бета-версии Mozilla Firefox (которые распространяются по таким каналам как Nightly, Beta, TestFlight и BuddyBuild) находятся в процессе разработки и могут иметь другие параметры конфиденциальности.
